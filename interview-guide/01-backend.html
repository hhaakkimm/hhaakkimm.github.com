gen = (x**2 for x in range(10))  # Memory efficient
  • Decorators: Function and class decorators, @property, @staticmethod, @classmethod 
    def timing_decorator(func):
    def wrapper(*args, **kwargs):
        start = time.time()
        result = func(*args, **kwargs)
        print(f"{func.__name__} took {time.time()-start}s")
        return result
    return wrapper
  • Context Managers: with statement, __enter__ and __exit__

    1. URL routing (URLconf)
    2. Middleware processing
    3. View execution
    4. Template rendering or JSON response
  • Django ORM:

users = User.objects.select_related('profile').prefetch_related('posts')
# Prevents N+1 queries

    • iterator() for large querysets
    • Database indexes on frequently queried fields
  • Caching: 
    from django.core.cache import cache
from django.views.decorators.cache import cache_page

@cache_page(60 * 15)  # Cache for 15 minutes
def vulnerability_list(request):
    data = cache.get('vulnerabilities')
    if not data:
        data = list(Vulnerability.objects.all())
        cache.set('vulnerabilities', data, timeout=900)
    return JsonResponse(data, safe=False)
